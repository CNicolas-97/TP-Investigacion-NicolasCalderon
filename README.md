# Trabajo Práctico: Trabajo de Investigación Conectividad de Sistemas y Bases de Datos

## Resumen del Proyecto
Este trabajo analiza los mecanismos de conexión de datos entre diversos lenguajes de programación y sistemas gestores de bases de datos. El enfoque central es la comprensión de las capas de comunicación (API, JDBC, ORM, HTTP) y la correcta categorización de las herramientas de desarrollo.

## Contenido Principal
- **Conectividad por Lenguaje:** Análisis de cómo C#, Java, Python, C++ y Node.js establecen puentes hacia los motores de datos.
- **Arquitectura de Herramientas:** Diferenciación entre motores persistentes (SQL), entornos de desarrollo (XAMPP), clientes GUI (HeidiSQL) y herramientas de simulación (JSON Server).
- **Implementación Práctica:** Ejemplos funcionales de conexión mediante ADO.NET, JDBC, DB-API y peticiones Fetch.

El código principal se encuentra en el archivo `index.html`.
